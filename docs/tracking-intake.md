# مدخل أحداث التتبع

المسار `POST /api/track/event` عام لأن صفحات الهبوط ترسل الحدث قبل تسجيل الدخول، لكنه محمي بمحدد طلبات مستقل ويتحقق من قائمة أحداث وحقول محددة فقط.

الحالة الحالية هي **استقبال بلا تخزين**: يعيد الخادم `202` بعد التحقق، ولا يحفظ أو يمرر أو يسجل حمولة الحدث. كما تُحذف الحقول غير المعروفة أثناء التحقق، ومنها أرقام الهاتف والبريد والرموز ومعرفات النقر وبيانات `UTM`. لا يجوز إضافة التخزين أو الربط بمنصة إعلانات قبل إضافة إدارة موافقة واضحة وسياسة احتفاظ بالبيانات.

الإعدادات الاختيارية:

- `TRACK_EVENT_RATE_LIMIT_WINDOW_MS`: نافذة التحديد، وافتراضيًا 60 ثانية.
- `TRACK_EVENT_RATE_LIMIT_MAX`: الحد لكل عنوان خلال النافذة، وافتراضيًا 120.
