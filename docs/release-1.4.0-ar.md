# الإصدار 1.4.0 — مركز الجوال واعتماد شريحة العمل

- إضافة واجهات الجوال الثابتة تحت `/api/mobile/v1` مع إبقاء `/api/gateway` للتوافق المؤقت.
- إنشاء مركز «الجوال والموظفون» لإسناد الجهاز والفرع واختيار شريحة العمل وتنفيذ طلب اتصال بعد موافقة الموظف.
- إنشاء رمز QR آمن مباشرة من مركز الجوال مع رابط الخادم ورمز يدوي بديل وصلاحية عشر دقائق لمرة واحدة.
- تجاهل الشريحة الشخصية أو المجهولة بالكامل قبل تسجيل المكالمة أو حفظ الرقم أو تشغيل واتساب.
- إضافة سياسات الرد: إيقاف، أرقام محددة، الجميع ما عدا، والجميع؛ مع إعادة فحص الاستثناء قبل الإرسال.
- إضافة تجربة واتساب حقيقية لمرة واحدة برقم اختياري، لا تدخل الطابور إذا كانت جلسة واتساب غير متصلة.
- إضافة صلاحيات جوال دقيقة قابلة للتخصيص لكل مستخدم، مع فصل صلاحية المسح والقفل.
- إضافة أوامر جوال محددة الأنواع، سجل تدقيق، انتهاء صلاحية، منع إعادة التشغيل، وإشعار فوري لا يحمل بيانات العميل.
- إضافة قاعدة بيانات للأجهزة والشرائح والأوامر والأحداث والصحة وسياسات الرد ونتائج الاختبارات.
- ربط المكالمات والمهام والصفقات والنتيجة بعد الاتصال باسم الموظف والجهاز.

يتطلب الإرسال الفوري إعداد Firebase على الخادم والتطبيق، ويتطلب الإرسال الحقيقي بواتساب تفعيل `WHATSAPP_PROVIDER=web` وبوابات السلامة الإنتاجية الحالية.
